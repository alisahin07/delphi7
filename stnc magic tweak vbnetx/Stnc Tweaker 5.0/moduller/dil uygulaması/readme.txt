This sample demostrates creating a localized form, changing the culture
and re-loading the form using the new culture.  East Asian languages 
require that the proper font is installed in order to display the characters correctly.
To install the language support on Windows 2000, Click on Start, Settings, Control Panel, Regional Options.
In the General Tab, see "Langauge settings for the system".  Check the boxes for
the language support you want to install.  In addition to Western European languages,
this sample displays forms in: Japanese, Korean, Chinese (Simplified) and Chinese (Traditional).  
Click OK after selecting the desired language support install options.  
You will have to restart your system to install the language support.
